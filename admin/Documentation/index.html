Introduction

Deped Manila Website is a responsive and clean HTML5 template. It is a template mainly used for Schools Division Manila with front page and admin side

Structure

CSS

Files are explained below:

Admin
File Description
sb-admin-2.css admin side css file. It is the main css file for Admin panel. It contains all the css code needed for the admin panel.
login.css Login css for login.php
Home
404.css This file contains the css code for the 404 page.
bootstrap.min.css Core Bootstrap CSS file for home page.
history.css history.php Bullet style
